Simple Button example:

```js
<Button
  backgroundcolor="aliceblue"
  boxcolor="black"
  boxcolorhover="blue"
  borderradius="4px"
  colorhover="blue"
  height="40px"
  transition="all 0.2s ease-in-out"
  width="100px">
  Hello world
</Button>
```
